## 這是我第一次學習框架、Restful API、SPA練習專案

## 題目是"TodoList"
前端
Vue.JS 2.x
後端
Laravel 8.x

功能 

1、事項CRUD

2、Login與signup

3、篩選已完成/未完成事項

4、編輯功能可直接點選該事項進行修改

## 下載/開啟專案

下載專案→開啟Terminal→cd至專案目錄中

<h3>前端</h3> 
確認Node.JS版本，請在Terminal輸入`node -v` 

確認是否有npm 客戶端，請在Terminal輸入`npm -v`

下載vue，`npm install vue`

執行`npm install`，將套件安裝回來

執行`npm run serve` 開啟專案

開啟Terminal→cd至專案目錄中
<h3>後端</h3>

下載composer

安裝laravel，執行`composer global require laravel/installer`

執行`php artisan serve`開啟server
